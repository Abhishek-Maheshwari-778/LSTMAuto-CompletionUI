# 🚀 AI Sentence Auto-Completion System
Developed by **Abhishek Maheshwari**

This is a deep learning project that uses **LSTM (Long Short-Term Memory)** networks to predict and auto-complete sentences. It features a modern, high-performance web UI built with Streamlit.

## ✨ Features
- **Real-time Prediction**: Generates multiple word completions as you type.
- **Customizable**: Adjust the number of predictions via a slider.
- **Premium UI**: Sleek dark-mode interface with custom CSS styling.
- **Pre-trained Model**: Uses a robust LSTM model trained on natural language datasets.

## 🛠️ Tech Stack
- **Backend**: Python, TensorFlow, Keras
- **Frontend**: Streamlit, Custom CSS
- **Model**: LSTM Neural Network

## 🚀 Quick Start

### 1. Install Dependencies
```bash
pip install tensorflow streamlit numpy Pillow
```

### 2. Run the App
```bash
streamlit run app.py
```
